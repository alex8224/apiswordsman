医务业务接口

患者信息

接口名称 患者信息
请求方式 GET
请求URL /api/patient

请求参数

参数名 类型 必填 描述
id String 是 患者ID

响应参数

参数名 类型 描述
name String 患者姓名
age Integer 患者年龄

调用样例

GET /api/patient?id=123 HTTP/1.1
Host: api.example.com

就诊记录

接口名称 就诊记录
请求方式 GET
请求URL /api/visit

请求参数

参数名 类型 必填 描述
patient_id String 是 患者ID

响应参数

参数名 类型 描述
visit_id String 就诊记录ID
date String 就诊日期

调用样例

GET /api/visit?patient_id=123 HTTP/1.1
Host: api.example.com
