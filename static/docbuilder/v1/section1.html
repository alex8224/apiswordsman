HRP接口方案

接口名称 HRP接口方案
请求方式 POST
请求URL /api/v1/hrp

请求参数

参数名 类型 必填 描述
param1 String 是 参数1描述
param2 Integer 否 参数2描述

响应参数

参数名 类型 描述
response1 String 响应参数1描述
response2 Integer 响应参数2描述

调用样例

POST /api/v1/hrp HTTP/1.1
Host: api.example.com
Content-Type: application/json

{
    "param1": "value1",
    "param2": 123
}

单点登录

接口名称 单点登录
请求方式 POST
请求URL /api/v1/sso

请求参数

参数名 类型 必填 描述
username String 是 用户名
password String 是 密码

响应参数

参数名 类型 描述
token String 认证Token
expires_in Integer 过期时间（秒）

调用样例

POST /api/v1/sso HTTP/1.1
Host: api.example.com
Content-Type: application/json

{
    "username": "user1",
    "password": "pass123"
}

用户管理

接口名称 用户管理
请求方式 GET
请求URL /api/v1/users

请求参数

参数名 类型 必填 描述
id String 否 用户ID

响应参数

参数名 类型 描述
users Array 用户列表

调用样例

GET /api/v1/users HTTP/1.1
Host: api.example.com

权限管理

接口名称 权限管理
请求方式 POST
请求URL /api/v1/permissions

请求参数

参数名 类型 必填 描述
user_id String 是 用户ID
permissions Array 是 权限列表

响应参数

参数名 类型 描述
status String 操作状态

调用样例

POST /api/v1/permissions HTTP/1.1
Host: api.example.com
Content-Type: application/json

{
    "user_id": "user1",
    "permissions": ["read", "write"]
}

系统设置

接口名称 系统设置
请求方式 GET
请求URL /api/v1/settings

请求参数

参数名 类型 必填 描述
key String 否 设置项的键

响应参数

参数名 类型 描述
settings Object 系统设置

调用样例

GET /api/v1/settings HTTP/1.1
Host: api.example.com
